# Fake News Detection

A machine learning application that predicts whether a news article is likely real or fake.

## Features

- News article analysis
- ML-based classification
- Confidence score
- Streamlit frontend

## Installation

```bash
git clone https://github.com/deepansharoraa/fake-news-detection.git
cd fake-news-detection

python -m venv venv
